## 1. NOVEDADES VERSIÓN 7
- **Interfaz "Light Mode":** Diseño de alto contraste (Blanco/Azul) reemplazando el tema oscuro.
- **Módulo Vendedores:** Gestión completa con foto de perfil.
- **Manejo de Imágenes:** Carga dinámica de fotos para Productos y Vendedores desde URL.
- **POS Mejorado:** Listas de selección reales, cálculo automático y validación de roles.
- **Arquitectura:** Inclusión de `utils/helpers.py` y `controllers/vendor_controller.py`.

## 2. INSTALACIÓN
1. `pip install -r requirements.txt` (Asegurar que `requests` está instalado).
2. `python populate_data.py` (Si la BD está vacía).
3. `python main.py`

## 3. MAPA DEL PROYECTO
- **Main:** `main.py` (Entry point).
- **Controladores:** Lógica pura SQL (Productos, Clientes, Ventas, Vendedores, Auth, Reportes).
- **Vistas:**
  - `pos_ui.py`: Facturación con listas dinámicas.
  - `vendors_ui.py`: ABM Vendedores + Foto.
  - `inventory_ui.py`: ABM Productos + Foto + Alertas Stock.
  - `main_window_ui.py`: Menu lateral y StackedWidget.
- **Assets:** `styles.py` (QSS de alto contraste).
